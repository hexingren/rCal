This is Hexing Ren's Calculator. Its name is rCal.

It implements basic mathematical operations.
It gives out error information when:
1. "error: indived by zero"
2. "error: malformed input" if the input is in {+, -, *, /} and the next input is in {+, -, *, /, =}
3. "error: out of bound" (-1000000, 1000000)
